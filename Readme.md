# Seeeeee:D 「PocketMentor」

## エレベータピッチ

**PocketMentor**は  
**いつかやりたいことが溜まること**を解決したい  
**ついついだらけちゃう人**向けの  
**モチベ支援アプリケーション**です。  
これは、**やりたいことに関連した記事を提案することで、やる気を回復させること**により、  
**TODOアプリ**とは違って  
**有意義な休日**を実現できます。
